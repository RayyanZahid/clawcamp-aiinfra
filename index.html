PREVIEW Up next ·failure modes

It all falls apart.

Breunig taxonomy · 2025

Four ways context dies.

01
Poison

A hallucination enters context and gets referenced forever. (Gemini played Pokémon and invented goals it couldn't achieve.)

02
Distract

Past ~100k, agents favor repeating history over inventing new plans. Memory beats reasoning.

03
Confuse

More tools = worse performance. Berkeley's function-calling leaderboard confirms it across every model.

04
Clash

Sharded prompts with conflicting guidance drop scores 39%. o3 fell 98 → 64.

Every failure mode is a context mode. Fix the context, the agent gets smarter for free.

PART 03 Honest start

Your AI
is not
super-intelligent.

Working with subagents is not
hard orchestration.

Comms between agents inherit
every human-comms problem.

So the harness has to do the heavy lifting.

RESET Three myths to drop

What you're actually
working with.

01 Not super-intelligence

It's pattern matching with a tool belt. Confidently wrong is the default, not the exception. The Replit agent didn't malfunction — it was being itself.

02 Not hard orchestration

Subagents are LLM calls in a loop with prompts as routing logic. No distributed runtime. No consensus protocol. Text in, text out, dispatched by code you wrote in an afternoon.

03 Not magic comms

Two agents talking is two flawed humans on Slack — ambiguity, context drop, telephone-game decay, conflicting decisions. Faster, and without the social cues that catch drift.

If you wouldn't trust a tired junior engineer
to run that command on prod,
don't trust the agent either.

Hardening, then, is not optional. It's the whole job.

EXHIBIT A The catastrophe tape

What goes wrong,
in production.

Six real incidents, 2024–2026. Different vendors, same shape: the safety property lived in the prompt, not in the harness.

Jul 2025 · Replit Wiped prod DB during a code freeze. 1,206 execs + 1,196 companies deleted. 4,000 fake users fabricated. "I panicked instead of thinking." Antidote — PreToolUse deny on DROP|DELETE against env=prod DSNs. Dev/prod isolation by IAM, not by prompt.
Apr 2026 · PocketOS Cursor + Opus 4.6 deleted the volume in 9 seconds. 30+ hour outage. Backups colocated with the volume they "protected." Customers stranded at car-rental pickups. Antidote — least privilege per-tool, not per-agent. Backups behind a separate IAM principal. Any cloud verb matching *Delete* is tier-3.
Dec 2025 · AWS Kiro "Cleanest fix" was to delete and rebuild prod. 13-hour AWS Cost Explorer outage in mainland China. Goal-drift from "minor bug" to "nuke environment." Antidote — IAM excludes *:Delete* on env=prod. Plans containing destructive verbs get a typed approver field.
Feb 2024 · Air Canada Chatbot invented a bereavement-fare refund. BCCRT ruled the airline liable: "the chatbot is not a separate legal entity." First common-law precedent. Antidote — typed verb QuotePolicy(id) returns canonical text. Money-back terms must cite a verifiable policy ID. Cite-or-refuse.
Feb 2025 · OpenAI Operator Bought $31.43 of groceries without consent. User asked for a price comparison. Operator ordered. The "ask before purchasing" guarantee was a soft prompt instruction. Antidote — approval_gate=True on every Verb.Purchase(). Implemented in the harness, not in the prompt.
Aug 2012 · Knight Capital $440M in 45 minutes. Pre-LLM. Same shape. Botched 8-server deploy left dead 2003 code live. Bought high, sold low across 154 stocks. ~397M shares. Antidote — every worker reports a version+spec hash. Orchestrator refuses to dispatch on drift. Per-strategy kill switch. PnL-drift circuit breaker.

PREVIEW Up next ·tier the action

Tier the verbs. Gate the tier.

HARDENING 01 Reversibility tiers

Classify every verb
by how loud it screams at 3 a.m.

T1 Read · Glob · Grep · Edit-in-cwd · npm test Reversible local. dontAsk + allowlist. No prompt. Git is your undo button.
T2 git commit · git branch · gh pr create · npm install Git-recoverable. ask rule for first run, allow after canary. Never --force.
T3 git push origin main · prod schema migration · vercel deploy --prod Visible to others. PR-only — agent opens, human merges. Deny direct push.
T4 DROP · DELETE FROM · Volume.Delete · Stripe.Refund · SendEmail Outbox + deterministic worker. HITL approval card. Idempotency key. Agent never holds the credentials.

Default deny on T3 and T4. Promote a verb a tier only after the canary passes for a week. Promotion is a code change, not a prompt.

HARDENING 02 Isolate the blast radius

One agent should not
hold every tool.

01 Orchestrator

Read-only. Plans, delegates, synthesizes. Read · Glob · Grep · Agent. Cannot mutate state. The orchestrator can never rm -rf.

02 Executor

Narrow writer. Tool allowlist scoped to one domain (edit, deploy, send). Inherits a fresh context window. permissionMode: dontAsk.

03 Verifier

Read-only judge. Runs evals, diffs, tests. Emits a binary verdict. Sees the executor's diff but not the orchestrator's plan.

Multi-agent Opus + Sonnet beat single-agent Opus by 90.2% on internal research evals. Anthropic · multi-agent research system
"Actions carry implicit decisions. Conflicting decisions carry bad results." Walden Yan · Cognition

Reconcile both: parallel reads, serialized writes.

Footgun · bypassPermissions on the orchestrator silently disarms every subagent. Never use it outside a sandbox.

HARDENING 03 Budgets & breakers

Errors compound.
Make the harness
trip the wire.

Real incident

$437 by 7 a.m.

A nightly summarizer entered a retry loop at 11 p.m. and racked up $437 on the company card before anyone noticed. No turn cap. No cost ceiling. No tool-level breaker.

Harness-enforced ceilings · TypeScript

const LIMITS = { maxTurns: 30, maxToolCalls: 100, maxInputTokens: 500_000, maxCostUSD: 5.00, }; for await (const m of query({ prompt, options: { permissionMode: "dontAsk" } })) { stats.cost += costOf(m.usage); if (stats.cost > LIMITS.maxCostUSD) throw new Error("cost ceiling"); }

Per-tool circuit breaker · PreToolUse

# 5 failures in 60s → trip # human resets a flag in Redis if failures_in_window(tool, 60) >= 5: return deny( "breaker open: " + tool + " — human reset required" ) # one probe after cooldown # avoid stampeding the upstream

"Agents are stateful and errors compound." · Anthropic

HARDENING 04 Evals · canaries · shadow

Don't measure attempts.
Measure all-k success.

pass@k
≥1 of k
pass^k
all of k

pass@k says the agent can.
pass^k says the agent does, every time. For critical workflows, ship on pass^k.

Source · Anthropic · "demystifying evals for ai agents"

The graduation ladder

  • Plan mode — agent emits the diff. Human reads. Apply by hand.
  • Dry-run — every destructive verb has a --preview twin that returns the would-be SQL/HTTP.
  • Shadow — agent decides, human ratifies. Log the delta. Graduate when delta is acceptable.
  • Canary — every deploy runs a hello-world that exercises every tool surface.
  • Production — only after the canary has stayed green for a week.

You're not training the model. You're training the harness around it.

PREVIEW Up next ·the lethal trifecta

Three corners. Remove one.

HARDENING 05 Adversarial defense

The PR title
is already
in your prompt.

01 · Private data Production secrets, customer PII, internal docs in the agent's context.
02 · Untrusted content PR titles, issue bodies, web pages, MCP tool descriptions, PDFs with white-on-white text.
03 · Egress Any tool that POSTs outbound. WebFetch. Files API. SendEmail. Even a markdown image renders.

Combine all three and the agent is exfiltratable. Remove any one corner and the trifecta breaks. — Simon Willison · Jun 2025

Real published exploits · 2025

  • Comment & Control · Oct 2025 · CVSS 9.4. PR title in the prompt → ANTHROPIC_API_KEY exfiltrated. Hit Claude Code, Gemini CLI, Copilot.
  • EchoLeak · CVE-2025-32711. Zero-click data theft from M365 Copilot via crafted email.
  • postmark-mcp · Sept 2025. First in-the-wild malicious MCP server. BCC field silently added; outgoing emails copied to attacker.
  • Notion AI exfil · Sept 2025. White-on-white PDF instructions. No user action.

Defenses you can ship today

  • Credential brokering · agent never holds prod tokens. Outbox + deterministic worker.
  • Egress allowlist · WebFetch(domain:...) only. Block curl/wget outright.
  • MCP server pinning · SHA-pin servers. Allowlist only. Disable auto-update.
  • Provenance tags · wrap external content in <<<UNTRUSTED>>> markers; strip chat-template literals.
  • Output filter · scrub secrets and external links from anything the agent emits.

HARDENING 06 The OpenClaw / Claude Code checklist

Twelve checkboxes.
Ship before you sleep.

For OpenClaw operators and Claude Agent SDK users running agents against critical systems. One pass through this list closes 80% of the catastrophe tape.

Audit · run openclaw security audit --deep and resolve every finding before first prod run.
Bind · gateway.bind: "loopback". Tailscale Serve over raw LAN. Never 0.0.0.0 unauthenticated.
Auth · gateway.auth.mode: "token". Long random secret. Separate from hooks.token.
Permission mode · dontAsk for headless. plan for diff-first. Never bypassPermissions outside a sandbox.
Allowlist · explicit allowed_tools. Default-deny groups automation, runtime, fs until needed.
Deny rules · git push --force, rm -rf, DROP|DELETE, ~/.aws/**, **/.env. Belt + braces with a PreToolUse hook.
Sandbox · sandbox mode "all" for non-main sessions. Workspace "ro" unless the verb genuinely needs to write.
Atomic commits · Stop hook auto-commits at end of every turn. SessionStart hook refuses dirty trees.
Loop budgets · maxTurns, maxToolCalls, maxCostUSD enforced in the harness. Trip on cost, not on apology.
Outbox · agent emits intent records, not DELETEs. Deterministic worker holds prod credentials and executes.
MCP pinning · allowlist only. SHA-pin every server. Disable auto-update. Postmark rug-pulled in 9 hours.
Shadow mode · first 100 prod runs: agent decides, human ratifies, log the delta. Graduate only on acceptable distribution.
